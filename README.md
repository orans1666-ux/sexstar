# صالون النجمة السادسة | Sixth Star Barbershop

مشروع Frontend كامل ومنظم لموقع صالون حلاقة رجالي فاخر باسم **صالون النجمة السادسة**. يعتمد على HTML5 وCSS3 وJavaScript فقط، مع تصميم متجاوب، دعم العربية والإنجليزية، RTL/LTR، الوضع الفاتح والداكن، حجز ودفع تجريبيين باستخدام Local Storage، ولوحة تحكم ثابتة ببيانات محاكاة.

## طريقة التشغيل

افتح الملف:

`index.html`

مباشرة من المتصفح. لا يحتاج المشروع إلى PHP أو Node.js أو أي Backend للتجربة.

## الهيدر الموحد

الهيدر الاحترافي موجود كتصميم CSS في `css/style.css` و `css/responsive.css`، ويتم ضمان ظهوره بشكل موحد في صفحات الواجهة العامة من خلال `js/main.js`. يحتوي على شعار الصالون، الاسم العربي والإنجليزي، روابط التنقل، زر الحجز، زر الدخول، زر اللغة، زر الوضع، وروابط واتساب وسناب شات.

## هيكل المشروع

- `index.html`: الصفحة الرئيسية مع الهيرو، الخدمات، التقييمات، البنرات، الخريطة، وروابط التواصل.
- `about.html`: قصة الصالون والرؤية والرسالة والقيم.
- `services.html`: جميع خدمات الصالون مع الأسعار والمدد وزر الحجز.
- `booking.html`: واجهة حجز تجريبية تحفظ البيانات في Local Storage.
- `login.html` و `register.html`: واجهات حساب تجريبية.
- `profile.html`: ملف العميل ونقاطه وحجوزاته.
- `my-bookings.html`: جدول الحجوزات التجريبية.
- `payment.html`: واجهة دفع تجريبية لا تحفظ بيانات البطاقة.
- `payment-success.html` و `payment-failed.html`: صفحات نتيجة الدفع.
- `contact.html`: معلومات التواصل والخريطة ونموذج رسالة.
- `promotions.html`: صفحة العروض والبنرات الإعلانية.
- `gallery.html`: معرض صور مصمم للصالون والخدمات والفريق.

## مجلدات التصميم والسكريبتات

- `css/style.css`: الهوية العامة، الهيدر، الأزرار، البطاقات، الأقسام، والفوتر.
- `css/pages.css`: الصفحات الداخلية، النماذج، الجداول، الحجز، الملف الشخصي، والمعرض.
- `css/banners.css`: البنرات الإعلانية والعروض والـ slider.
- `css/dashboard.css`: تصميم لوحة التحكم.
- `css/responsive.css`: تجاوب الجوال والتابلت.

- `js/main.js`: بيانات الخدمات والتقييمات والتفاعل العام وقائمة الجوال.
- `js/language.js`: تبديل اللغة وحفظها وتغيير الاتجاه RTL/LTR.
- `js/theme.js`: تبديل الوضع الفاتح والداكن وحفظه.
- `js/booking.js`: محاكاة الحجز وحفظه في Local Storage.
- `js/auth.js`: محاكاة تسجيل الدخول وإنشاء الحساب والملف الشخصي.
- `js/payment.js`: محاكاة الدفع دون حفظ بيانات البطاقة.
- `js/banners.js`: تدوير البنرات في الصفحة الرئيسية.
- `js/gallery.js`: تصفية معرض الصور.
- `js/dashboard.js` و `js/analytics.js`: بيانات وإحصائيات لوحة التحكم.

## لوحة التحكم

مجلد `admin/` يحتوي على:

- `dashboard.html`
- `services.html`
- `bookings.html`
- `users.html`
- `payments.html`
- `reviews.html`
- `analytics.html`
- `settings.html`

اللوحة تتضمن Sidebar، Topbar، بطاقات KPI، جداول، أزرار، فلاتر، وتحليلات تجريبية.

## البنرات والصور

البنرات موجودة داخل:

`assets/images/banners/`

وتشمل:

- بنر رئيسي
- خصم 20% على أول زيارة
- باقة العريس
- تهذيب اللحية
- خدمات الشعر
- عروض نهاية الأسبوع
- الحجز عبر واتساب
- الدفع السهل

صور الخدمات موجودة في `assets/images/services/`، وصور المعرض في `assets/images/gallery/`، وصورة الهيرو في `assets/images/hero/`.

## تعديل روابط التواصل

- رابط Google Maps موجود داخل صفحات `index.html` و `contact.html` في عنصر `iframe`. غيّر قيمة `src` إلى رابط موقع الصالون الحقيقي.
- رابط WhatsApp الافتراضي هو `https://wa.me/966500000000`. يمكن تغييره في الصفحات أو داخل `js/main.js` للهيدر الموحد.
- رابط Snapchat الافتراضي هو `https://www.snapchat.com/`. يمكن تغييره في الصفحات أو داخل `js/main.js`.
- رقم الاتصال التجريبي هو `966500000000` ويمكن استبداله برقم الصالون الحقيقي.

## قاعدة البيانات

ملف MySQL موجود في:

`database/sixth_star_barbershop.sql`

ويحتوي على إنشاء قاعدة البيانات والجداول:

`users`, `services`, `barbers`, `bookings`, `payments`, `reviews`, `contact_messages`, `loyalty_points`, `settings`, `working_hours`, `holidays`

مع بيانات تجريبية مناسبة.

## ملاحظات مهمة

هذا المشروع Frontend فقط. JavaScript لا يتصل مباشرة بـ MySQL، وهذا مقصود. بيانات الحجز والحساب والدفع التجريبية تحفظ داخل Local Storage للمحاكاة فقط.

أي ربط حقيقي مع MySQL يحتاج Backend لاحقًا مثل API مخصص، مع حماية كلمات المرور، الجلسات، صلاحيات المستخدمين، ومعالجة الدفع من مزود دفع فعلي.
